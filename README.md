# 1984

## 简介

新 mac 前端开发环境初始化。

### 功能一览

- 自动监测是否安装 `brew`、 `node`、 `XCode` …… 未安装的自动安装。

## 安装

#### 方法一（推荐）
拷贝下面一行到你的 shell 下执行。

```bash
sh <(curl -s https://david-codingserf.github.io/1984/index.html)
```

#### 方法二
点击下载脚本 [start.sh](https://github.com/David-CodingSerf/1984/blob/master/start.sh)，然后执行。
